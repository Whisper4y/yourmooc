# yourmooc
在线教育平台--计算机课程学习网站，此网站的作用是提升自己对SSM框架的熟悉度。

所用技术：

数据库：MySQL

后端：Maven、Spring、SpringMVC、Mybatis

前端：Bootstrap、jQuery
